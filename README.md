
## About
A modern revamp of an older XML/Lua based in-game guide for the Neurotrauma Medical Overhaul for Barotrauma.

## Features
* Lua based automatically-building GUI replacing the older, more limited, XML variant;
* On-page 'hyperlinks' to allow for easy navigation between pages;
* A search bar, to make finding specific afflictions considerably easier;
* Integration into the Health Scanner / Hematology Analyzer to automatically open up pages for afflictions printed in chat if you click on them and a page exists;

## Want to contribute Localization?
By all means, do so! All language gets automatically updated based on the Localization XML Files. Simply take one of those, translate everything that is not within [brackets] or <part.of.the.code> and send a PR!
